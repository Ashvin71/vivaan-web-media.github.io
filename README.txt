Vivaan Web Media Website

This is a responsive business website for **Vivaan Web Media**, built to showcase web design, SEO, digital marketing, branding, commercial photography, product photography, and food photography services.

## About the Project

The website was created as a modern static website using front-end web technologies. It is designed for small businesses, growing brands, and corporate organizations that need a strong online presence.

## Built With

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- UI/UX Design
- GitHub Pages
- AI-assisted development tools
- Adobe Photoshop / Illustrator for creative support
- Commercial and product photography assets

## Features

- Responsive layout for desktop and mobile
- Business service sections
- Portfolio/gallery section
- Commercial photography showcase
- Product and food photography showcase
- Service area section
- Contact section
- SEO-friendly page title and meta description

## Services Highlighted
